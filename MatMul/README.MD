Для умножения матрицц на CPU использовался простыой тройной цикл.
Для GPU каждая нить вычисляет один элемент результирующей матрицы. Нить с индексом ij вычисляет элемент С(i, j)

Чистое время вычисления на GPU намного меньше, чем на CPU, и ускорение растёт с увеличением размера матрицы. Но для N = 100 ускорение с учётом пересылки данных CPU -> GPU почти равно еденице.
Ускорение для больших значений N растёт, что означает, что большие вычисления выгоднее по времени производить на GPU.

Также не стоит не забывать про инициализацию CUDA. При первом вызове ядра инициализируется драйвер, компилируется JIT-код.

Запуск проекта:

```bash
    # Сборка 
    docker compose build

    # Команда для запуска
    docker run --rm --gpus all \ -v ${PWD}:/workspace \ cuda_dev:latest \ bash -c "nvcc -O2 -arch=sm_86 /workspace/main.cu -o /tmp matmul && /tmp/matmul"
```

**Необходима видеокарта NVIDIA**
