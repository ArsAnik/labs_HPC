### Описание проделанной работы
Для вычисления числа Пи методом Монте-Карло на CPU использовался цикл, в котором генерировались случайные кооридинаты x и y с помощью rand(). Считалось количество точек, попавших в круг, затем вычислялось итоговое значение Пи.

На GPU расчёт похож, но для получения количества точек используется алгоритм параллельной редукции. 
Генерация чисел происходит прямо на GPU, поэтому единственным копированием результата на CPU можно принебречь.
Генератор случайных чисел curand инициализируются один раз до начала основного цикла, так как он достаточно долгий по времени (92.506050ms).

### Выводы
> При увеличении количества точек видно сильное ускорение расчётов на GPU по сравнению с CPU.

> Важно инициализировать curand до расчётов, иначе большое время инициализации добавится к времени работы.


| N | Время работы на CPU, ms | Время работы на GPU, ms | Ускорение |
| - | - | - | - |
| 1000 | 0.011000 | 0.106208 | 0.103570 |
| 1000000 | 8.382000 | 0.162784 | 51.491547 |
| 1000000000 | 8710.791016 | 38.109344 | 228.573624 |

Запуск проекта:

```bash
    # Сборка 
    docker compose build

    # Команда для запуска
    docker run --rm --gpus all \ -v ${PWD}:/workspace \ cuda_dev:latest \ bash -c "nvcc -O2 -arch=sm_86 /workspace/main.cu -o /tmp matmul && /tmp/matmul"
```

**Необходима видеокарта NVIDIA**
