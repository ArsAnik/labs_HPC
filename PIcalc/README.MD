### Описание проделанной работы
Для вычисления числа Пи методом Монте-Карло на CPU использовался цикл, в котором генерировались случайные кооридинаты x и y с помощью rand(). Считалось количество точек, попавших в круг, затем вычислялось итоговое значение Пи.

На GPU расчёт похож, но для получения количества точек используется алгоритм параллельной редукции. 
Генератор случайных чисел CURAND инициализируются один раз до начала основного цикла, чтобы не включать дорогостоящую инициализацию в замер времени.


### Выводы
> Время вычислений на GPU стабильно меньше времени вычичлений на CPU. (при N > 100000)

Запуск проекта:

```bash
    # Сборка 
    docker compose build

    # Команда для запуска
    docker run --rm --gpus all \ -v ${PWD}:/workspace \ cuda_dev:latest \ bash -c "nvcc -O2 -arch=sm_86 /workspace/main.cu -o /tmp matmul && /tmp/matmul"
```

**Необходима видеокарта NVIDIA**
