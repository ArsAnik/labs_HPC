### Описание проделанной работы

В данной работе реализован генетический алгоритм на GPU для аппроксимации функции полиномом степени 4.

Начальная популяция генерируется случайным образом. Далее алгоритм выполняет итерации, в которых происходят следующие шаги генетического алгоритма:

> Fitness - вычисляется как сумма квадратов ошибок между исходной функцией и аппроксимирующим полиномом на заданных точках.   

> Crossover - новые особи путём комбинирования параметров родителей. Используется поиск лучшего родителя и расчёт взвешенного среднего для каждого коэффициента ребёнка.  

> Mutation - добавление случайного шума к коэффициентам. Изменение случайного бита оказалось менее эффективным для получения лучшего результата.  

> Снова происходит расчёт показателя Fitness.  

> Selection - используется библиотека thrust для сортировки массива по увеличению fitness.  


Цикл останавливается либо при достижении максимального количества поколений (итераций), либо при достижении целевого значения приспособленности (fitness).

Генерация случайных чисел производится с помощью библиотеки CURAND. Инициализация занимает около 11 секунд.

Сама работа алгоритма в среднем занимает **5,17 сек (5170 ms)**

Запуск проекта:

```bash
    # Сборка 
    docker compose build

    # Команда для запуска
    docker run --rm --gpus all \ -v ${PWD}:/workspace \ cuda_dev:latest \ bash -c "nvcc -O2 -arch=sm_86 /workspace/main.cu -o /tmp matmul && /tmp/matmul"
```

**Необходима видеокарта NVIDIA**
