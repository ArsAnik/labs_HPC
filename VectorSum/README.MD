### Описание проделанной работы
Для вычисления суммы элементов вектора на CPU использовался простой последовательный цикл суммы.

На GPU реализован алгоритм параллельной редукции. Каждая нить загружает элемент из входного вектора в общую память блока, блоки выполняют локальную редукцию и частичные суммы блоков записываются. 
Затем в последующих проходах цикла частичные суммы суммируются по той же логике, что и изначальный вектор, до тех пор, пока не получится едиственный элемент, который и будет ответом.

Можно также заметить, что копирование вектора с CPU на GPU происходит достаточно медленно. Это связано с большим размером вектора. Для ускорения этого процесса массив на CPU был создан с помощью cudaMallocHost, а не malloc.

Запуск проекта:

```bash
    # Сборка 
    docker compose build

    # Команда для запуска
    docker run --rm --gpus all \ -v ${PWD}:/workspace \ cuda_dev:latest \ bash -c "nvcc -O2 -arch=sm_86 /workspace/main.cu -o /tmp matmul && /tmp/matmul"
```

**Необходима видеокарта NVIDIA**
